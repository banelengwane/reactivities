# Reactivities
- This is a social media-like Web application, where users register and log in to share what activities they are about to do with their friends.
- The registered users will then indicate the type of activity, the date and the venue. And their friends can follow these activities, which they can attend later.
- All users following the same event get notified when there is a change in the event, and have a chat section where they can share views about the event.

## Technologies used:
- React 18 
- Typescript
- .Net 8 (Entity framework, CQRS, Mediator Pattern, SignalR)
- SQLite
